# SiWarga - Sistem Informasi RT/RW

SiWarga adalah aplikasi sistem informasi RT/RW berbasis Laravel untuk membantu pengurus mengelola data warga, RT, kartu keluarga, iuran, kas, pengajuan surat, pengaduan, UMKM, pengumuman, kegiatan, backup data, dan log aktivitas.

## Fitur Utama

- Landing page publik
- Login multi-role: Admin RW, Ketua RT, Warga
- Kelola data RT
- Kelola data warga
- Kelola kartu keluarga
- Import dan export data warga
- Kelola pengumuman
- Kelola kegiatan
- Kelola UMKM warga
- Pengaduan warga dengan kode tracking
- Pengajuan surat warga dengan cetak PDF
- Kelola jenis surat
- Iuran warga
- Kas masuk dan kas keluar
- Dashboard admin dengan filter periode
- Export laporan dashboard PDF
- Backup dan restore database
- Log aktivitas
- Pencarian global admin
- Panel Ketua RT
- Panel Warga

## Tech Stack

- Laravel 12
- PHP 8.2
- MySQL / MariaDB
- Blade
- Tailwind CSS
- DomPDF
- XAMPP
